# PSX-IntelliTrade

A stock analysis and intelligence system for the Pakistan Stock Exchange (PSX).
The project focuses on fetching market data, analyzing trends, and assisting
users in making informed trading decisions using data-driven insights.

---

## Project Overview

PSX-IntelliTrade is designed to:
- Fetch PSX stock data
- Analyze historical price movements
- Generate insights using technical indicators
- Visualize trends and outputs
- Serve as a foundation for intelligent trading systems

This project is primarily built for academic and learning purposes.

---

## Tech Stack

- Python
- Flask (Backend / API)
- Yfinance (Data Fetching)
- Pandas & NumPy (Data Processing)
- Plotly (Visualization)
- Streamlit (Frontend / Dashboard)
