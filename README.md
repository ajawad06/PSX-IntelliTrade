# PSX-IntelliTrade

A stock analysis and intelligence system for the Pakistan Stock Exchange (PSX).
The project focuses on fetching market data, analyzing trends, and assisting
users in making informed trading decisions using data-driven insights.

---

## Tech Stack

- Python
- Flask (Backend / API)
- Yfinance (Data Fetching)
- Pandas & NumPy (Data Processing)
- Plotly (Visualization)
- Streamlit (Frontend / Dashboard)
